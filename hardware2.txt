MDDS60 (RC MODE 1 1 0 0 1 1) - 모터 드라이버
Pixhawk 1 기본 - 픽스호크
PI5 Model B - 라즈베리파이
IYP-24V10Ah-A3 - 배터리팩


* 공통사항
배터리 → 퓨즈 → 전원 스위치 → MDDS60 / DC-DC / Power Module

PowerModule → Pixhawk POWER IN
DC-DC (5V 3~5A급) → PI 5V PIN2
DC-DC GND → PI GND (9번)
* DC to DC 컨버터는 24V to 5V 사용
전원 스위치 → MDDS60 V+ (24V)
전원 스위치 → MDDS60 V- (GND)

PI GPIO 14 (TX) PIN8 → Pixhawk TELEM2 RX (3번)
PI GPIO 15 (RX) PIN10 → Pixhawk TELEM2 TX (2번)
PI GND PIN9 → Pixhawk TELEM2 GND (6번)

PI GPIO 23 PIN16 → 조향 서보 모터 Signal (주황/노랑/흰색)
DC-DC 5V BEC → 조향 서보 모터 +5V 전원 (빨강)
DC-DC GND → 조향 서보 모터 GND (갈색/검정)
* DC to DC 컨버터는 24V to 5V 사용
* 5V 4A~5A 이상 컨버터를 사용 시 Pi + 서보 1개 동시 사용 가능

Pixhawk MAIN OUT CH1 S → MDDS60 RC1 Ω (Signal)
Pixhawk MAIN OUT CH1 - → MDDS60 RC1 - (GND)

MDDS60 MOTOR 1 A → 뒤쪽 DC 모터 +
MDDS60 MOTOR 1 B → 뒤쪽 DC 모터 –
* DC 모터는 극성이 없으므로 A/B 반대로 꽂아도 동작하며, 방향만 반대가 됨.

Pixhawk MAIN OUT CH2 S → MDDS60 RC2 Ω (Signal)
Pixhawk MAIN OUT CH2 - → MDDS60 RC2 - (GND)
MDDS60 MOTOR 2 A → 앞쪽 DC 모터 +
MDDS60 MOTOR 2 B → 앞쪽 DC 모터 –
* 앞/뒤 모터는 되도록 동일 스펙 모터 사용 권장

Pixhawk 파라미터(2모터 동시 스로틀용)

SERVO1_FUNCTION = 70 (Throttle)
SERVO2_FUNCTION = 70 (Throttle)
* CH1 → MOTOR1, CH2 → MOTOR2 동일한 스로틀 명령 출력
